# VR/AR 头显最佳虚拟屏幕距离与尺寸速查表

> 基于现有光学数据和厂商信息整理的虚拟屏幕最佳观看参数速查表。
>
> **核心原则：** 头显的虚像距离（固定焦距）是多少，虚拟屏幕就放在多远，以实现调节-辐辏零冲突。
>
> **观影场景：** 按影院感统一设定 **45° 水平视角**，宽高比 **16:9**。
>
> **计算公式：**
> - 宽度 = 2 × 距离 × tan(22.5°) ≈ 距离 × **0.828**
> - 高度 = 宽度 × 9/16 ≈ 宽度 × **0.5625**
> - 对角线英寸 ≈ 宽度(米) × **49.5**（16:9 系数）
>
> 用于 OpenXR 场景虚拟屏幕距离与尺寸设计参考，需在 GUI 中增加 VR 头显型号选择项以自动设置对应参数。

## 使用要点

- 上表尺寸能营造 **IMAX 影院般的视野覆盖感**，同时仍处于舒适视域内，无需频繁转头。
- 若头显软件中无法直接输入米，可按 `距离 × 0.828` 估算宽度，再换算为软件内的"屏幕尺寸"（通常以对角线英寸表示）：**对角线英寸 ≈ 宽度(米) × 49.5**（16:9 系数）。
- AR 眼镜因无完美遮挡，环境光干扰下建议适当缩小尺寸或降低背景亮度以增强对比。
- 如需介于办公与观影之间的折中尺寸，只需将公式中的系数 `0.828` 替换为 `0.73`（对应 **40°** 水平视角）即可。

## 核心使用说明与免责

1. **"推荐距离"并非绝对的单一数字**：厂商的虚像距离标注常有 ±0.1~0.2 米误差。实际应用中，软件会将这个距离模糊成一个舒适区间。设为 1.3 米或 1.5 米对眼脑来说，都属于"高匹配区"，无需纠结到厘米级。

2. **硬件厂商的混用标注**：部分厂商可能标注"等效 3 米观看 XX 寸大屏"，这是**大小/视场角的等效**，不等于**光学虚像距离**。本表只讨论光学虚像距离。

3. **AR 眼镜光波导方案**：表中 AR 眼镜多为 Birdbath 方案。若是**光波导信息提示类眼镜**（如 INMO、OPPO Air Glass、早期的 Google Glass），虚像距离通常在 **无穷远（∞）或 4~6 米**，同样建议直接推到远处。

4. **自行验证方法**：在一个纯色空旷的虚拟空间里（无透视背景干扰），不断推远虚拟屏幕，如果到某个距离后，**文字的锯齿感、晶状体的紧张感突然消失且进入一种放松的平视状态**，那个距离就是头显的真实光学焦点附近。

### 水平视角调节

GUI 中应提供水平视角滑块（可选，默认 **45°**）：

| 视角 | 系数 (2×tan(θ/2)) | 适用场景 |
| :--- | :--- | :--- |
| 30° | 0.536 | 办公、多窗口 |
| 35° | 0.631 | 折中 |
| **40°** | **0.728** | 介于办公与观影之间 |
| **45°** | **0.828** | **影院沉浸（默认）** |
| 50° | 0.934 | 更大覆盖感 |

### 配置项

- **自动检测**：通过 OpenXR runtime 获取头显型号（如 `XrSystemProperties` 中的 `systemName`），自动匹配上表。
- **自定义**：允许用户手动输入距离（范围 0.5~40 米），宽度和高度随之自动计算。
- **默认值**：检测不到时默认 1.3 米（覆盖最广泛的菲涅尔焦距），配 45° 视角。
